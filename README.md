# AI-Enhanced Capstone Project

A frontend AI engineering capstone built with modern web technologies. This repository serves as the foundation for designing, developing, and deploying an AI-enhanced user experience.

## Project Overview

This project is part of the FlyRank AI frontend engineering capstone. It focuses on building a responsive, modular web application that integrates AI-driven features into a production-ready frontend workflow.

The application is designed to demonstrate:

- Clean, component-based architecture using React and Next.js
- Responsive UI built with Tailwind CSS
- Practical integration patterns for AI-assisted user experiences
- Maintainable code conventions suitable for team collaboration

> **Status:** Early-stage repository. Core application scaffolding and feature implementation are in progress.

## Prerequisites

Before you begin, ensure the following are installed:

- **Node.js** — v18.x or later (LTS recommended)
- **npm**, **yarn**, **pnpm**, or **bun** — package manager of your choice
- **Git** — for cloning and version control

## Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/<your-username>/<repository-name>.git
   cd Assignment01
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

   > Use `yarn`, `pnpm`, or `bun` if that is your preferred package manager.

3. **Configure environment variables**

   Create a `.env.local` file in the project root:

   ```env
   # Add required environment variables here
   # NEXT_PUBLIC_API_URL=
   # OPENAI_API_KEY=
   ```

4. **Start the development server**

   ```bash
   npm run dev
   ```

5. **Open the application**

   Visit [http://localhost:3000](http://localhost:3000) in your browser.
